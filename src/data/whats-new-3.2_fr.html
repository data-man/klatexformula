Nouvelle bibliothèque

Le fonctionnement interne de la bibliothèque a été complètement revu. La bibliothèque incorpore maintenant une structure de « ressources », c'est-à-dire une source (abstraite) de formules latex, qui peuvent physiquement résider dans la bibliothèque interne, dans un fichier base de données, dans un ficher d'exportation de bibliothèque de KLatexFormula 3.1, ou éventuellement dans d'autres endroits.
